# Plan de Acción Financiero Integral — Repo Operativo

Este repositorio implementa en archivos accionables el **plan de 90 días** para pasar de deuda a rentabilidad y construir ingresos recurrentes. Está organizado para ejecutar *quick wins*, convertirlos en *retainers* y escalar con tu LLC (Wyoming) y marcas white‑label (Knotie‑AI, LancePilot, Climbo).

## Estructura
- `docs/` → Roadmap 30/60/90, ofertas, KPIs, validación, cumplimiento LLC.
- `sales/` → guiones, emails, propuesta base.
- `automation/` → plantillas y paquetes: LancePilot (WhatsApp), Knotie (Voice AI), Climbo (reputación).
- `ops/` → CRM simple (CSV) y operativa de co‑venta con tu hermano.
- `finance/` → flujo de caja semanal (CSV), AR/DSO, plantilla de factura.
- `legal/` → acuerdo de reparto de ingresos (rev‑share) y checklist BOI/IRS.
- `.github/` → templates de issues/PR y workflows.
- `project/` → guía para crear el Project/kanban en GitHub.

## Uso rápido
```bash
# 1) Crea el repo local y copia este contenido
git init
git add .
git commit -m "bootstrap: repo operativo del plan financiero 90d"

# 2) Crea un repo en GitHub (reemplaza URL)
git branch -M main
git remote add origin https://github.com/USER/plan-accion-financiero-integral.git
git push -u origin main
```

### Tablero (kanban) sugerido
- **Backlog** → Ideas y oportunidades
- **Esta semana** → Compromisos con fecha
- **En curso** → Entrega/producción
- **Revisión** → QA/cliente
- **Hecho** → Cerrar con retro y métricas

> Importante: registra *cada* lead y pago; mide KPIs semanales (flujo de caja, AR/DSO, tasa de cierre) y conviértelo en hábito operacional.
