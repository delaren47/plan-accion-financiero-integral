## How to use

- **This is an intake, not a backlog.** Items land here raw. Promote each into this repo's own working files (`NEXT_ACTIONS.md` / `PENDIENTES.md` / daily standup / decision log), then move it to `## Processed`.
- **Append-only — never delete.** Handled items go to `## Processed` with a date + 1-line outcome.
- **Line format:** `- YYYY-MM-DD | <one-liner> | <source / [[wikilink]]>`
- **Routing tag of record:** `[PAFI]` — the router matches this tag in personal-os to route here.
