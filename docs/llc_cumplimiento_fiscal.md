# LLC (Wyoming) — Cumplimiento y Flujo

- **BOI (FinCEN)** obligatorio.
- **EIN** para banca y facturación.
- Facturar todo por la LLC (incl. MX/CA).
- Bancos: Mercury / Relay.
- Anuales: **5472 + 1120 pro forma** (partes relacionadas), **1040‑NR** si aplica.
- Trabajar con CPA especializado en no‑residentes.
