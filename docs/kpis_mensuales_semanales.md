# KPIs — Finanzas, Ventas, Operación

**Semanales (finanzas):**
- Flujo de caja semanal
- Booking (ingresos facturados)
- AR y DSO (objetivo < 15 días)

**Ventas/marketing:**
- Leads calificados, tasa de cierre (>25%)
- LTV, CAC

**Mensuales (operación):**
- MRR, Churn (<5%)
- Margen bruto por servicio (>70%)
